# 🌡️ Temperature Converter

A clean and interactive Temperature Converter that converts between Celsius, Fahrenheit, and Kelvin in real-time.

🎯 **Live Demo:** [Visit App](https://temperature-converter-zubair.vercel.app)

---

## ✨ Features
- Real-time conversion between Celsius, Fahrenheit and Kelvin
- Type in any field and others update automatically
- Clear/Reset button
- Shows formula used for conversion
- Clean modern dark theme UI
- Fully responsive design
